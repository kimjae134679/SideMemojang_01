## v1.0.0

### 핵심 기능
- 메모 무제한 추가 / 탭 전환
- 패널 접힘 / 펼침 (단축키, 트레이, 클릭)
- 드래그 핸들로 세로 위치 조절
- 모서리 핸들로 패널 크기 실시간 조절
- 좌/우 붙이기 전환
- 멀티 모니터 이동 지원
- 시스템 트레이 아이콘

### 편집기
- 굵게 / 기울임 / 밑줄 / 취소선 (Ctrl+B/I/U, Ctrl+Shift+X)
- 글씨 색상 팔레트
- `->` → `→` 자동 변환
- `- + Space` → `•` 자동 변환
- Tab 키: 포커스 이동 없이 공백 3칸 삽입
- Ctrl+Z / Ctrl+Y 메모별 실행 취소 / 다시 실행

### 설정 / UX
- 24가지 색상 테마 + 채도 슬라이더
- 항상 위에 / 다른 곳 클릭 시 접기 / 오타 확인 옵션
- Windows 시작 시 자동 실행
- 메모 데이터 내보내기 / 가져오기 / 초기화
- 180ms 디바운스 자동 저장
- 삭제 후 5초 실행 취소 토스트
